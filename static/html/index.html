Home page

HTTP METHODS Path Description
GET / (Home page) Retrieves the home page of the website

Users

HTTP METHODS Path Description
GET /users Retrieves the home page of the website
GET /users/:id This endpoint retrieves one user, specified by it's id
POST /users/register This endpoint registers a new user, hash's their password and adds them to the db.
PATCH/PUT /users/:id This endpoint updates one user, specified by it's id
DELETE /users/:id This endpoint deletes one user, specified by it's id

Product/Item

HTTP METHODS Path Description
GET /products This endpoint retrieves all products
GET /products/:id This endpoint retrieves one product, specified by it's id
POST products/add This endpoint add a product to the db.
PUT/PATCH products/:id This endpoint updates one product, specified by it's id
DELETE products/:id This endpoint deletes one product, specified by it's id
